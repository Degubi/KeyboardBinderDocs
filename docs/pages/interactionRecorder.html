Interaction Recorder

To ease the process of creating more complicated mouse/keyboard interaction based scripts it is recommended to use the Interaction recorder.


Starting

The recording process is really simple: You start the recorder and you get some generated script source for mimicing the recorded interactions.
To start the recording process select the 'Interaction Recorder' option from the application's tray icon menu.
A small overlay box will open on the top left side of the screen.




Usage

Using the recorder is really easy:

  • To start recording middle click on the mouse
  • Every mouse click, mouse movement and keyboard interaction gets recorded along with the delay between interactions
  • To stop recording middle click on the mouse
  • After a successful recording a desktop notification will appear with some information
  • The generated script source gets copied to the clipboard. To use it paste it into a user script file.

While recording interactions the overlay screen logs the interactions.




Example output

It is worth mentioning that the generated script is usually not optimal. It is recommended to clean up before using it.
Here's an example output of a recording session:

Mouse.set_cursor_location(712, 233)
Mouse.click(Mouse.BUTTON_LEFT)
Keyboard.type('ASD')
Common.wait(2460)
Mouse.set_cursor_location(712, 233)
Mouse.click(Mouse.BUTTON_LEFT)
Common.wait(518)
Mouse.set_cursor_location(1012, 246)
Mouse.click(Mouse.BUTTON_RIGHT)
Common.wait(912)
Mouse.set_cursor_location(821, 219)
Mouse.click(Mouse.BUTTON_LEFT)
Keyboard.type('VVVV')
