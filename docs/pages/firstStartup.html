Installing & First Startup


Installing & Running

  • Unzip KeyboardBinder.zip into a folder
  • Start KeyboardBinder.exe

On the first startup Keyboard Binder will handle some setup related stuff.
After completing the setup it will start running in the background. To know if it is running you can check the application's green keyboard tray icon:


Setup

To start seting up Keyboard Binder for actual usage select 'Settings' from the right click menu of the application's tray icon.
