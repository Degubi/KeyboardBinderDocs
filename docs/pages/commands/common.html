Common

Contains commands for generic tasks like waiting & interacting with the user's clipboard

copySelectionToClipboard()

Description:

Copies the currently selected text to the clipboard

Example:

Common.copySelectionToClipboard()

copyTextToClipboard(text: string)

Description:

Copies the given text to the clipboard

Example:

Common.copyTextToClipboard(text = 'epic text')

exec(command: string)

Description:

Executes the given terminal command with cmd

Example:

Common.exec(command = 'explorer https://google.com')

waitMs(time: integer)

Description:

Delays the next command execution by the given amount of milliseconds

Example:

Common.waitMs(time = 250)
