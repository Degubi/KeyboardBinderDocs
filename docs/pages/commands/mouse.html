Mouse

click(button: enum)

Description:

Presses the specified button of the mouse

Example:

Mouse.click(button = right)

moveAbs(x: integer, y: integer)

Description:

Moves the mouse cursor to the specified absolute x and y positions

Example:

Mouse.moveAbs(x = 50, y = 50)

moveRel(x: integer, y: integer)

Description:

Moves the mouse cursor by the specified x and y positions relative to the current position of the cursor

Example:

Mouse.moveRel(x = 50, y = 50)
