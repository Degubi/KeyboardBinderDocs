OBS module

Contains functions used to interact with OBS


Importing

from keyboardBinder import OBS

Constants

EVENT_CURRENT_SCENE_CHANGED | EVENT_OBS_EXIT


Functions

add_event_listener(event: OBS.EVENT_OBS_EXIT, on_exit: () -> None) -> None

Description:

Adds a listener function to be called when OBS is shutting down

Example:

OBS.add_event_listener(event = OBS.EVENT_OBS_EXIT, on_exit = lambda: print('We closin'))

add_event_listener(event: OBS.EVENT_CURRENT_SCENE_CHANGED, on_change: (new_scene_name: str) -> None) -> None

Description:

Adds a listener function to be called when the current scene changes. The first function argument is the new scene's name

Example:

OBS.add_event_listener(event = OBS.EVENT_CURRENT_SCENE_CHANGED, on_change = lambda scene_name: print('Scene name changed'))

get_current_scene_name() -> str

Description:

Gets the current scene's name

Example:

scene_name = OBS.get_current_scene_name()

get_input_volume(input_name: str) -> int

Description:

Returns the volume of the given input in decibels

Example:

volume = OBS.get_input_volume(input_name = 'intro-song')

get_is_input_muted(input_name: str) -> bool

Description:

Returns whether the given input is muted

Example:

is_muted = OBS.get_is_input_muted(input_name = 'outro-song')

pause_media_input(media_input_name: str) -> None

Description:

Pauses the given media input

Example:

OBS.pause_media_input(media_input_name = 'outro-music')

play_media_input(media_input_name: str) -> None

Description:

Example:

OBS.play_media_input(media_input_name = 'outro-music')

restart_media_input(media_input_name: str) -> None

Description:

Restars the given media input

Example:

OBS.restart_media_input(media_input_name = 'outro-music')

set_current_scene(scene_name: str) -> None

Description:
