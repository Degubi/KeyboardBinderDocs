OBS module

Contains functions used to interact with OBS


Importing

from keyboardBinder import OBS

Constants

EVENT_OBS_EXIT

MEDIA_PAUSE | MEDIA_PLAY | MEDIA_RESTART | MEDIA_STOP


Functions

add_event_listener(event: OBS.EVENT_OBS_EXIT, on_exit: () -> None) -> None

Description:

Adds a listener function to be called when OBS is shutting down

Example:

OBS.add_event_listener(event = OBS.MEDIA_PLAY, on_exit = lambda: print('We closin'))

get_input_volume(input_name: str) -> int

Description:

Returns the volume of the given input in decibels

Example:

OBS.get_input_volume(input_name = 'MISSING_ARG_VALUE')

get_is_input_muted(input_name: str) -> bool

Description:

Returns whether the given input is muted

Example:

OBS.get_is_input_muted(input_name = 'MISSING_ARG_VALUE')

set_input_is_muted(input_name: str, muted: bool) -> None

Description:

Sets the specified input's mute state to the given value

Example:

OBS.set_input_is_muted(input_name = 'epic-frag-song', muted = False)

set_input_volume(input_name: str, volume: int) -> None

Description:

Sets the specified input's volume to the given volume in decibels

Example:

OBS.set_input_volume(input_name = 'MISSING_ARG_VALUE', volume = 'MISSING_ARG_VALUE')

set_media_input_state(media_input_name: str, state: OBS.MEDIA_PLAY | OBS.MEDIA_PAUSE | OBS.MEDIA_STOP | OBS.MEDIA_RESTART) -> None

Description:

Sets the state of the given media input

Example:

OBS.set_media_input_state(media_input_name = 'outro-music', state = OBS.MEDIA_PLAY)

start_recording() -> None

Description:

Starts recording

Example:

OBS.start_recording()

stop_recording() -> None

Description:

Stops recording

Example:

OBS.stop_recording()
