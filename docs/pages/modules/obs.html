OBS

Contains functions used to interact with OBS


add_on_exit_listener(listener: () -> None) -> None

Description:

Adds a listener function to be called when OBS is shutting down

Example:

OBS.add_on_exit_listener(listener = lambda: print('OBS exit!'))

restart_media_source(source_name: str) -> None

Description:

Restarts the specified media source

Example:

OBS.restart_media_source(source_name = 'background-music')

set_media_source_paused(source_name: str, paused: bool) -> None

Description:

Plays or pauses the give media source based on the given value

Example:

OBS.set_media_source_paused(source_name = 'outro-music', paused = True)

set_source_muted(source_name: str, muted: bool) -> None

Description:

Sets the specified media source's mute state to the given value

Example:

OBS.set_source_muted(source_name = 'epic-frag-song', muted = False)

start_recording() -> None

Description:

Starts recording

Example:

OBS.start_recording()

stop_media_source(source_name: str) -> None

Description:

Stops the given media source

Example:

OBS.stop_media_source(source_name = 'background-music')

toggle_media_source_pause(source_name: str) -> None

Description:

Plays or pauses the given media source based on whether it is currently playing or not (toggling it)

Example:

OBS.toggle_media_source_pause(source_name = 'background-music')

toggle_source_mute(source_name: str) -> None

Description:

Toggles the specified meda source's mute state

Example:

OBS.toggle_source_mute(source_name = 'background-music')
