Keyboard

Contains functions used to interact with the keyboard


MODIFIER_SHIFT = 16


hold_modifier_key(modifier_key: 16) -> None

Description:

Holds the given key on the keyboard. The key is not released until a Keyboard.release call

Example:

Keyboard.hold_modifier_key(modifier_key = 16)

release_modifier_key(modifier_key: 16) -> None

Description:

Releases the given key on the keyboard

Example:

Keyboard.release_modifier_key(modifier_key = 16)

type(text: str) -> None

Description:

Types the given text on the keyboard

Example:

Keyboard.type(text = 'hi team')
