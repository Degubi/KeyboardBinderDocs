Keyboard module

Contains functions used to interact with the keyboard


Importing

from keyboardBinder import Keyboard

Constants

MODIFIER_ALT | MODIFIER_CTRL | MODIFIER_SHIFT


Functions

hold_modifier_key(modifier_key: Keyboard.MODIFIER_SHIFT | Keyboard.MODIFIER_CTRL | Keyboard.MODIFIER_ALT) -> None

Description:

Holds the given key on the keyboard. The key is not released until a Keyboard.release call

Example:

Keyboard.hold_modifier_key(modifier_key = Keyboard.MODIFIER_SHIFT)

is_alt_key_down() -> bool

Description:

Returns true if the alt key is being held down

Example:

is_down = Keyboard.is_alt_key_down()

is_ctrl_key_down() -> bool

Description:

Returns true if the ctrl key is being held down

Example:

is_down = Keyboard.is_ctrl_key_down()

is_shift_key_down() -> bool

Description:

Returns true if the shift key is being held down

Example:

is_down = Keyboard.is_shift_key_down()

release_modifier_key(modifier_key: Keyboard.MODIFIER_SHIFT | Keyboard.MODIFIER_CTRL | Keyboard.MODIFIER_ALT) -> None

Description:

Releases the given key on the keyboard

Example:

Keyboard.release_modifier_key(modifier_key = Keyboard.MODIFIER_SHIFT)

type(text: str) -> None

Description:

Types the given text on the keyboard

Example:

Keyboard.type(text = 'hi team')

while_holding_modifier_key(modifier_key: Keyboard.MODIFIER_SHIFT | Keyboard.MODIFIER_CTRL | Keyboard.MODIFIER_ALT, action: () -> None) -> None

Description:

Runs the given action while holding the passed in key

Example:

Keyboard.while_holding_modifier_key(modifier_key = Keyboard.MODIFIER_CTRL, action = lambda: Keyboard.type('S'))
