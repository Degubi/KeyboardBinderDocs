Keyboard module

Contains functions used to interact with the keyboard


Importing

from keyboardBinder import Keyboard

Constants

KEY_ALT | KEY_CTRL | KEY_SHIFT


Functions

hold_modifier_key(modifiers: Keyboard.KEY_CTRL | Keyboard.KEY_SHIFT | Keyboard.KEY_ALT) -> None

Description:

Holds the given key(s) on the keyboard. The key(s) is/are not released until a Keyboard.release call

Example:

Keyboard.hold_modifier_key(modifiers = Keyboard.KEY_CTRL)

is_modifier_key_down(modifiers: Keyboard.KEY_CTRL | Keyboard.KEY_SHIFT | Keyboard.KEY_ALT) -> bool

Description:

Returns true if the given modifier key is being held down

Example:

is_down = Keyboard.is_modifier_key_down(modifiers = Keyboard.KEY_CTRL)

release_modifier_key(modifiers: Keyboard.KEY_CTRL | Keyboard.KEY_SHIFT | Keyboard.KEY_ALT) -> None

Description:

Releases the given key(s) on the keyboard

Example:

Keyboard.release_modifier_key(modifiers = Keyboard.KEY_CTRL)

type(text: str) -> None

Description:

Types the given text on the keyboard

Example:

Keyboard.type(text = 'hi team')

while_holding_modifier_key(modifiers: Keyboard.KEY_CTRL | Keyboard.KEY_SHIFT | Keyboard.KEY_ALT, action: () -> None) -> None

Description:

Runs the given action while holding the passed in key(s)

Example:

Keyboard.while_holding_modifier_key(modifiers = Keyboard.KEY_SHIFT, action = lambda: Keyboard.type('S'))
