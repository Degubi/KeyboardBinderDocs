VLC module

Contains functions used to interact with VLC


Importing

from keyboardBinder import VLC

Functions

jump_to_next_item() -> None

Description:

Jumps to the next item in the current playlist

Example:

VLC.jump_to_next_item()

jump_to_previous_item() -> None

Description:

Jumps to the previous item in the current playlist

Example:

VLC.jump_to_previous_item()

pause_playback() -> None

Description:

Pauses playing the current item

Example:

VLC.pause_playback()

resume_playback() -> None

Description:

Resumes playing the current item

Example:

VLC.resume_playback()

stop_playback() -> None

Description:

Stops playing the current item

Example:

VLC.stop_playback()
