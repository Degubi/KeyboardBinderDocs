Mouse module

Contains functions used to interact with the mouse


Importing

from keyboardBinder import Mouse

Constants

BUTTON_LEFT | BUTTON_MIDDLE | BUTTON_RIGHT


Functions

click(button: Mouse.BUTTON_LEFT | Mouse.BUTTON_MIDDLE | Mouse.BUTTON_RIGHT) -> None

Description:

Presses the specified button of the mouse

Example:

Mouse.click(button = Mouse.BUTTON_LEFT)

get_cursor_location() -> CursorLocation

Description:

Returns a tuple where the first value is the x, and the second one is the y location of the mouse cursor

Example:

x, y = Mouse.get_cursor_location()

hold(button: Mouse.BUTTON_LEFT | Mouse.BUTTON_MIDDLE | Mouse.BUTTON_RIGHT) -> None

Description:

Presses (but never releases) the specified button of the mouse

Example:

Mouse.hold(button = Mouse.BUTTON_RIGHT)

release(button: Mouse.BUTTON_LEFT | Mouse.BUTTON_MIDDLE | Mouse.BUTTON_RIGHT) -> None

Description:

Releases the specified button of the mouse

Example:

Mouse.release(button = Mouse.BUTTON_RIGHT)

set_cursor_location(x: int, y: int) -> None

Description:

Moves the mouse cursor to the specified absolute x and y location

Example:

Mouse.set_cursor_location(x = 50, y = 50)
