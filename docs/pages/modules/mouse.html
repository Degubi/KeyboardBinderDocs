Mouse

Contains functions used to interact with the mouse


BUTTON_LEFT = 1024

BUTTON_MIDDLE = 2048

BUTTON_RIGHT = 4096


click(button: Mouse.BUTTON_LEFT | Mouse.BUTTON_MIDDLE | Mouse.BUTTON_RIGHT) -> None

Description:

Presses the specified button of the mouse

Example:

Mouse.click(button = Mouse.BUTTON_LEFT)

get_cursor_location() -> CursorLocation

Description:

Returns a tuple where the first value is the x, and the second one is the y location of the mouse cursor

Example:

x, y = Mouse.get_cursor_location()

move_cursor_to(x: int, y: int) -> None

Description:

Moves the mouse cursor to the specified absolute x and y location

Example:

Mouse.move_cursor_to(x = 50, y = 50)
