Desktop module

Contains functions for generic Desktop related tasks


Importing

from keyboardBinder import Desktop

Functions

copy_selection_to_clipboard() -> None

Description:

Copies the currently selected text to the clipboard

Example:

Desktop.copy_selection_to_clipboard()

copy_text_to_clipboard(text: str) -> None

Description:

Copies the given text to the clipboard

Example:

Desktop.copy_text_to_clipboard(text = 'epic text')

exec_command(command: str) -> None

Description:

Executes the given terminal command with cmd

Example:

Desktop.exec_command(command = 'explorer https://google.com')

get_master_volume_level() -> int

Description:

Returns the master volume level in percentage

Example:

volume_level = Desktop.get_master_volume_level()

get_process_volume_level(process_name: str) -> int

Description:

Returns the volume level of the target process in percentage. If the given process is not active -1 gets returned.

Example:

volume_level = Desktop.get_process_volume_level(process_name = 'ts3client_win64.exe')

get_selected_file_paths_in_file_browser() -> list[str]

Description:

Returns the paths of all selected files in an active file browser window

Example:

file_paths = Desktop.get_selected_file_paths_in_file_browser()

move_file_to_trash(file_path: str) -> None

Description:

Moves the given file to the trash bin

Example:

Desktop.move_file_to_trash(file_path = 'pic_of_ex.png')

set_master_volume_level(value: int) -> None

Description:

Sets the master volume level to the given percentage value

Example:

Desktop.set_master_volume_level(value = 50)

set_process_volume_level(process_name: str, value: int) -> None

Description:

Sets the volume level of the target process to the given percentage value. If the given process is not active nothing gets set.

Example:

Desktop.set_process_volume_level(process_name = 'ts3client_win64.exe', value = 50)
