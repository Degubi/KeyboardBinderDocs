Common module

Contains functions for generic tasks like waiting & interacting with the user's clipboard


Importing

from keyboardBinder import Common

Functions

copy_selection_to_clipboard() -> None

Description:

Copies the currently selected text to the clipboard

Example:

Common.copy_selection_to_clipboard()

copy_text_to_clipboard(text: str) -> None

Description:

Copies the given text to the clipboard

Example:

Common.copy_text_to_clipboard(text = 'epic text')

exec_command(command: str) -> None

Description:

Executes the given terminal command with cmd

Example:

Common.exec_command(command = 'explorer https://google.com')

wait(seconds: float) -> None

Description:

Delays the next command execution by the given amount of milliseconds

Example:

Common.wait(seconds = 3.5)
