Application module

Contains functions for interacting with KeyboardBinder itself. Most of the functionality in this module are available in the user tray menu


Importing

from keyboardBinder import Application

Constants

APP_DESKTOP | APP_NOTEPADPP | APP_OBS | APP_PREMIERE_PRO


Functions

block_keyboard_input(app_name: Application.APP_PREMIERE_PRO | Application.APP_NOTEPADPP | Application.APP_DESKTOP) -> bool

Description:

Blocks the keyboard input for the given application. Returns true if the blocking attempt was successful or it was already blocked, false otherwise

Example:

blocking_successful = Application.block_keyboard_input(app_name = Application.APP_PREMIERE_PRO)

connect_to_application(app_name: Application.APP_PREMIERE_PRO | Application.APP_OBS) -> bool

Description:

Connects to the given application. Returns true if the connection attempt was successful or it was already connected, false otherwise

Example:

connection_successful = Application.connect_to_application(app_name = Application.APP_PREMIERE_PRO)

disable_console() -> None

Description:

Disables the console window

Example:

Application.disable_console()

enable_console() -> None

Description:

Enables the console window

Example:

Application.enable_console()

get_active_profiles() -> list[str]

Description:

Returns the list of active profile names

Example:

active_profile_names = Application.get_active_profiles()

is_connected_to_application(app_name: Application.APP_PREMIERE_PRO | Application.APP_OBS) -> bool

Description:

Returns true if there's an active connection to the given application

Example:

is_connected = Application.is_connected_to_application(app_name = Application.APP_PREMIERE_PRO)

is_console_enabled() -> bool

Description:

Returns whether the console window is enabled

Example:

visible = Application.is_console_enabled()

open_docs_page() -> None

Description:

Opens the docs page in the browser

Example:

Application.open_docs_page()

show_settings_screen() -> None

Description:

Shows the settings screen

Example:

Application.show_settings_screen()

start_interaction_recorder(result_consumer: (recording_result: str) -> None) -> None

Description:

Starts the interaction recorder utility

Example:

Application.start_interaction_recorder(result_consumer = lambda result: print('Use result here'))

wait(seconds: float) -> None

Description:

Delays the next command execution by the given amount of milliseconds

Example:

Application.wait(seconds = 3.5)
