Application module

Contains functions for interacting with KeyboardBinder itself. Some of these are considered as developer functions, because most of these are available in the user tray menu


Importing

from keyboardBinder import Application

Functions

is_debug_console_visible() -> bool

Description:

Returns whether the debugger console window is visible

Example:

visible = Application.is_debug_console_visible()

open_docs_page() -> None

Description:

Opens the docs page in the browser

Example:

Application.open_docs_page()

set_debug_console_visible(visible: bool) -> None

Description:

Hides/show the debugger console window

Example:

Application.set_debug_console_visible(visible = True)

start_interaction_recorder(result_consumer: (recording_result: str) -> None) -> None

Description:

Starts the interaction recorder utility

Example:

Application.start_interaction_recorder(result_consumer = lambda result: print('Use result here'))

wait(seconds: float) -> None

Description:

Delays the next command execution by the given amount of milliseconds

Example:

Application.wait(seconds = 3.5)
