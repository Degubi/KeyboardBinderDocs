Application module

Contains functions for interacting with KeyboardBinder itself. Most of the functionality in this module are available in the user tray menu


Importing

from keyboardBinder import Application

Constants

APP_PREMIERE_PRO


Functions

connect_to_application(app_name: APP_PREMIERE_PRO) -> bool

Description:

Connects to the given application. Returns true if the connection attempt was successful, false otherwise

Example:

successful = Application.connect_to_application(app_name = Application.APP_PREMIERE_PRO)

disable_debug_console() -> None

Description:

Disables the debugger console window

Example:

Application.disable_debug_console()

enable_debug_console() -> None

Description:

Enables the debugger console window

Example:

Application.enable_debug_console()

is_debug_console_enabled() -> bool

Description:

Returns whether the debugger console window is enabled

Example:

visible = Application.is_debug_console_enabled()

open_docs_page() -> None

Description:

Opens the docs page in the browser

Example:

Application.open_docs_page()

start_interaction_recorder(result_consumer: (recording_result: str) -> None) -> None

Description:

Starts the interaction recorder utility

Example:

Application.start_interaction_recorder(result_consumer = lambda result: print('Use result here'))

wait(seconds: float) -> None

Description:

Delays the next command execution by the given amount of milliseconds

Example:

Application.wait(seconds = 3.5)
