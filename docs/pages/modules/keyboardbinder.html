KeyboardBinder module

Contains functions for interacting with Keyboard Binder itself


Importing

from keyboardBinder import KeyboardBinder

Functions

disable_console() -> None

Description:

Disables the console window

Example:

KeyboardBinder.disable_console()

enable_console() -> None

Description:

Enables the console window

Example:

KeyboardBinder.enable_console()

get_active_profiles() -> list[str]

Description:

Returns the list of active profile names

Example:

active_profile_names = KeyboardBinder.get_active_profiles()

inject_into_process(process_name: str, block_keyboard_input: bool, on_process_exit: () -> None) -> bool

Description:

Useful for loads of stuff. Returns true if the injection attempt was successful or it was already injected, false otherwise

Example:

injection_successful = KeyboardBinder.inject_into_process(process_name = 'notepad++.exe', block_keyboard_input = True, on_process_exit = lambda result: print('Process exited!'))

is_console_enabled() -> bool

Description:

Returns whether the console window is enabled

Example:

visible = KeyboardBinder.is_console_enabled()

open_docs_page() -> None

Description:

Opens the docs page in the browser

Example:

KeyboardBinder.open_docs_page()

show_settings_screen() -> None

Description:

Shows the settings screen

Example:

KeyboardBinder.show_settings_screen()

shutdown() -> None

Description:

Shuts down Keyboard Binder

Example:

KeyboardBinder.shutdown()

start_interaction_recorder(result_consumer: (recording_result: str) -> None) -> None

Description:

Starts the interaction recorder utility

Example:

KeyboardBinder.start_interaction_recorder(result_consumer = lambda result: print('Use result here'))

wait(seconds: float) -> None

Description:

Delays the next command execution by the given amount of milliseconds

Example:

KeyboardBinder.wait(seconds = 3.5)
