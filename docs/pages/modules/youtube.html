Youtube module

Contains functions used to interact with Youtube


Importing

from keyboardBinder import Youtube

Functions

get_player_position() -> float

Description:

Returns the player's position in seconds

Example:

position_seconds = Youtube.get_player_position()

get_player_volume() -> int

Description:

Returns the player's volume in percentage

Example:

volume_level = Youtube.get_player_volume()

jump_to_next_video() -> None

Description:

Jumps to the next video in the playlist

Example:

Youtube.jump_to_next_video()

jump_to_previous_video() -> None

Description:

Jumps to the previous video in the playlist

Example:

Youtube.jump_to_previous_video()

load_video_by_id(id: str) -> None

Description:

Loads a video by the given ID

Example:

Youtube.load_video_by_id(id = 'dQw4w9WgXcQ')

pause_player() -> None

Description:

Pauses the player

Example:

Youtube.pause_player()

set_player_position(position_seconds: float) -> None

Description:

Sets the player's position to the given position in seconds

Example:

Youtube.set_player_position(position_seconds = 69)

set_player_volume(volume_level: int) -> None

Description:

Sets the player's volume to the given percentage value

Example:

Youtube.set_player_volume(volume_level = 42)

start_player() -> None

Description:

Pauses the player

Example:

Youtube.start_player()

start_server() -> None

Description:

Starts the server so that browser tabs can connect to Keyboard Binder

Example:

Youtube.start_server()

stop_player() -> None

Description:

Stops the player

Example:

Youtube.stop_player()

stop_server() -> None

Description:

Stops the server so that browser tabs no longer can connect to Keyboard Binder

Example:

Youtube.stop_server()
