Youtube module

Contains functions used to interact with Youtube


Importing

from keyboardBinder import Youtube

Functions

pause_player() -> None

Description:

Pauses the player

Example:

Youtube.pause_player()

start_player() -> None

Description:

Pauses the player

Example:

Youtube.start_player()

start_server() -> None

Description:

Starts the server so that browser tabs can connect to Keyboard Binder

Example:

Youtube.start_server()

stop_player() -> None

Description:

Stops the player

Example:

Youtube.stop_player()

stop_server() -> None

Description:

Stops the server so that browser tabs no longer can connect to Keyboard Binder

Example:

Youtube.stop_server()
