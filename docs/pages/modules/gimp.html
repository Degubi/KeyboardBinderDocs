Gimp module

Contains functions used to interact with Gimp


Importing

from keyboardBinder import Gimp

Functions

apply_brightness_contrast_filter(brightness: float, contrast: float, layer: LayerHandle) -> None

Description:

Applies the Brightness-Contrast filter with the given values to the given image

Example:

Gimp.apply_brightness_contrast_filter(brightness = 0.5, contrast = 0.25, layer = layer)

create_new_image(width: int, height: int) -> ImageHandle

Description:

Creates a new image with the given width and height

Example:

image = Gimp.create_new_image(width = 128, height = 128)

create_new_layer(name: str, width: int, height: int, opacity: int, image: ImageHandle) -> LayerHandle

Description:

Creates a new layer with the given name, width, height and opacity

Example:

layer = Gimp.create_new_layer(name = 'layer1', width = 128, height = 128, opacity = 50, image = image)

display_image(image: ImageHandle) -> None

Description:

Displays the given image

Example:

Gimp.display_image(image = image)

get_active_layer(image: ImageHandle) -> LayerHandle

Description:

Returns the active layer

Example:

layer = Gimp.get_active_layer(image = image)

get_image_size(image: ImageHandle) -> Dimension

Description:

Returns the size of the image

Example:

width, height = Gimp.get_image_size(image = image)

insert_layer(layer: LayerHandle, image: ImageHandle) -> None

Description:

Inserts the given layer into the given image

Example:

Gimp.insert_layer(layer = layer, image = image)

list_displayed_images() -> list[ImageHandle]

Description:

Returns the list of currently displayed images

Example:

images = Gimp.list_displayed_images()

load_image(path: str) -> ImageHandle

Description:

Loads an image from the given path

Example:

image = Gimp.load_image(path = 'cute_cat.jpg')
