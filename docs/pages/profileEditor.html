Profile Editor

After creating a new profile the editor always auto opens.
If you want to open the editor for an existing profile, click on the name of the profile in the previously shown profiles menu on the tray.

Profile Settings

On the left panel you can pick the profile's activator button (deactivator button is always F12),
the set of modifiers, an optional application to block the normal user input in and the profile's bound keyboard.

Let's start creating binds, to switch to a key's editor click on the target key.
