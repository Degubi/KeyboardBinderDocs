User Scripting

To start scripting naviage to a profile's script folder. It is recommended use an editor like VSCode for a better scripting experience.

It is also recommended to setup type checking inside the editor for convenience & avoiding some common errors.


Script File Structure

User scripts are written as regular python scripts.
There are a number of provided modules for interacting with the user's mouse/keyboard or other supported applications.


Simple Script Example

Let's start by creating a very simple script for e.g the 'A' key. To begin create a file named 'A.py' (in the profile's folder).
Paste in the following code:

from keyboardBinder import Mouse

Mouse.move_cursor_to(x = 50, y = 50)

The first line imports the Mouse module from the keyboardBinder's library. We use this module to interact with the user's mouse.
The second line moves the cursor to the given location on the screen. (the top left corner of the screen is considered 0 0)
For all the supported modules & functions check the 'Scripting Modules' section.


Running

Running a script is really easy:

  • Activate the profile with it's activation button on it's keyboard
  • Press the 'A' key on the profile's keyboard

__events__.py

This script file contains functions that get called on different lifecycle events of the profile.
These functions can be used to e.g connect to other applications or to setup input blocking, or to do clean up.

To learn how to block keyboard input check the Keyboard Input Blocking page.
