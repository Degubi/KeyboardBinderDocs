Application Configuration

To start configuring the application right click on the application's green keyboard tray icon:




Keyboards

First we need to add the keyboard(s) that we intend to use for scripting

  • Open up the 'Keyboards' menu option and click on the 'Add New Keyboard' option
  • Enter the name that you intend to use for the keyboard
  • Follow the instructions on the screen to finish adding the keyboard

Profiles

After adding a keyboard the next step is to create a profile

  • Open up the 'Profiles' menu option and click on the 'Create New Profile' option
  • Fill out the profile's parameters

A profile's parameters are:

  • A user defined name
  • A user picked keyboard
  • A user picked keyboard key that activates the profile ranging from F1 to F11
  • An optional application where the keyboard's input gets blocked (blocking normal input, running only scripts)

Profiles are stored in the application's 'scripts' folder, in this folder each subfolder represents a profile.
