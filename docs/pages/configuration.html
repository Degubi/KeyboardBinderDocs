Configuration

To configure the application use the desktop tray icon:

From this menu you can add keyboards, profiles, manage connections or close the application.
The architecture of the application is based around profiles.
A profile's parameters are:

  • A user defined name
  • A user picked keyboard key that activates the profile ranging from F1 to F11
  • A user picked keyboard
  • An optional application where the input gets blocked on the selected keyboard (blocking normal input, running only scripts)

Profiles are stored in the application's 'scripts' folder, in this folder each subfolder represents a profile.
