OBS Integration

OBS integration is implemented through the OBS Websocket plugin (this is available by default since v28)


Plugin Setup

  • In OBS go to Tools/obs-websocket Settings
  • Tick the 'Enable Websocket server' option
Note: It is also recommended to 'Enable System Tray Alerts' so that you know if you can actually connect to it.


Connecting

from keyboardBinder import Application

Application.connect_to_application('OBS')
