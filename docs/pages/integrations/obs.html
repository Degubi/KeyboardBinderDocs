OBS Integration

Connecting

from keyboardBinder import Application

Application.connect_to_application(Application.APP_OBS)

OBS integration is implemented through the OBS Websocket plugin. First make sure it is enabled in OBS settings.
To enable it tick the 'Enable Websocket server' option. It is also recommended to 'Enable System Tray Alerts' so that you know if you can actually connect to it.
