VLC Integration


VLC Setup

  • In VLC go to Tools/Preferences
  • Switch to 'All' settings from 'Simple' on the bottom left corner
  • Navigate to Interface/Main interfaces (or search for 'Main interfaces')
  • Tick in the 'Web' option
  • Navigate to Interface/Main interfaces/Lua (or search for 'Lua')
  • Set a password in the 'Lua HTTP' panel (this is required by VLC)
  • Set the same password in Keyboard Binder's integration settings
