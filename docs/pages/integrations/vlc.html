VLC Integration


VLC Setup

  • In VLC go to Tools/Preferences
  • Switch to 'All' settings from 'Simple' on the bottom left corner
  • Navigate to Interface/Main interfaces (or search for 'Main interfaces')
  • Tick in the 'Web' option
  • Navigate to Interface/Main interfaces/Lua (or search for 'Lua')
  • Set a password in the 'Lua HTTP' panel (this is required by VLC)
  • Set the same password in Keyboard Binder's integration settings


Controlling Playback

In VLC there are 3 playback states: Playing, Paused, Stopped.
To switch between these states there are 4 functions to do so:

from keyboardBinder import VLC

# to start playing the current playlist item
VLC.start_playback()

# to pause playing the current playlist item
VLC.pause_playback()

# to stop playing the current playlist item
VLC.stop_playback()


Controlling Player Volume

from keyboardBinder import VLC

# increase player volume by 15%
VLC.set_player_volume(VLC.get_player_volume() + 15)


Controlling The Playlist

VLC uses a single playlist as the item queue for playback.
You can check this in VLC by right clicking and checking out the 'Playlist' menu.
For controlling the playlist there are several functions:

from keyboardBinder import VLC

# navigate playlist
VLC.jump_to_previous_playlist_item()
VLC.jump_to_next_playlist_item()

# query playlist
playlist_item_paths = VLC.get_playlist_items()

# update playlist
VLC.add_item_to_playlist('Path to mp3 file')
