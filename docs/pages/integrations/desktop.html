Desktop Integration


Volume API

In a normal desktop environment there are 2 types of volume values: master system volume & per process volume.
There are separate functions for getting/setting master or per process volume values:

from keyboardBinder import Desktop

# increase master volume level by 10%

master_volume_level_percentage = Desktop.get_master_volume_level()
Desktop.set_master_volume_level(master_volume_level_percentage + 10)

# decrease chrome's volume level by 5%

chrome_volume_level_percentage = Desktop.get_process_volume_level('chrome.exe')
Desktop.set_process_volume_level('chrome.exe', chrome_volume_level_percentage - 5)

Clipboard API

from keyboardBinder import Desktop

# set clipboard text
Desktop.set_clipboard_text('CSGO')

# get clipboard text (clipboard_text = 'CSGO')
clipboard_text = Desktop.get_clipboard_text()

# basically CTRL + C
Desktop.copy_selection_to_clipboard()

# basically CTRL + V
Desktop.paste_clipboard_content()
