Premiere Pro Integration

Premiere Pro integration is implemented through a custom extension panel.


Extension Setup

  • In Keyboard Binder go to Settings/Integrations/Premiere Pro
  • Make sure the 'Application Directory' path is correct
    (the default installation directory is already here, if you have installed Premiere Pro elsewhere you have to put it's root directory here)
  • If a correct path is detected the input field should turn green
  • Click the Install button
  • Open up Premiere Pro and activate the extension by clicking on Window/Extensions/KeyboardBinder Panel
Note: The extension only works if this panel is active


Connecting

from keyboardBinder import Application

Application.connect_to_application('Premiere Pro')

Note: you have to call this function once per session, so calling it on profile activation is a perfect place.


Presets

To start applying presets to your clips first you have to import your presets into Keyboard Binder.
To import them first you have to find the path of your presets file (usually found in Documents/Adobe/Premiere Pro/14.0/Profile-Name/Effect Presets and Custom Items.prfpset).
This is the default path for presets. With this path you have to call the import_presets_from_file function.

Note: you have to call this function once per session, so calling it on profile activation is a perfect place.

After importing the presets file they can be applied to the current selection with the apply_audio_preset_to_selection or the apply_video_preset_to_selection functions.

from keyboardBinder import PremierePro

# Somewhere in profile activation code
PremierePro.import_presets_from_file('.prfpset file path')

# Somewhere else on keypress
PremierePro.apply_video_preset_to_selection('Presets/PresetNameHere')
