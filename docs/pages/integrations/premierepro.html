Premiere Pro Integration

Premiere Pro integration is implemented through a custom extension panel


Setup

  • In Keyboard Binder go to Settings/Integrations/Premiere Pro
  • Make sure the 'Application Directory' path is correct
    (the default installation directory is already here, if you have installed Premiere Pro elsewhere you have to put it's root directory here)
  • If a correct path is detected the input field should turn green
  • Click the Install button
  • Open up Premiere Pro and activate the extension by clicking on Window/Extensions/KeyboardBinder Panel
Note: The extension only works if this panel is active


Presets

Presets can be applied to the current selection with the apply_audio_preset_to_selection or the apply_video_preset_to_selection functions.

from keyboardBinder import PremierePro

PremierePro.apply_video_preset_to_selection('Presets/PresetNameHere')
