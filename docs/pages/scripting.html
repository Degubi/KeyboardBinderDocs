Scripting

To create a script naviage to a profile's script folder.
It is recommended to open this folder in an editor like VSCode for a better scripting experience.
Bind scripts are written as python scripts. To create a script for e.g the 'A' key create a file named 'A.py'.
Here's an example for a very boring, but simple script:

from keyboardBinder import Mouse

print(Mouse.move_cursor_to(x = 50, y = 50))

This simple code moves the cursor to the given location on the screen.
To run this script activate the profile with the profile's activation button on the selected keyboard and press the 'A' key.
For all the supported functions check the 'Modules' section.
