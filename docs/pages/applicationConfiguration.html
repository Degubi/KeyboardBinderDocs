Application Configuration

To start configuring the application select 'Settings' from the right click menu of the application's green keyboard tray icon:




Keyboards

First we need to add the keyboard(s) that we intend to use for scripting:

  • Select the 'Keyboards' tab on the top
  • Enter the keyboard's name
  • Select a physical keyboard by following the instructions on the screen after clicking on the 'Pick Physical Keyboard' button
  • Click the 'Add' button on the right side

Profiles

After adding a keyboard the next step is to create a profile:

  • Select 'Profiles' tab on the top
  • Enter the profile's name
  • Select a keyboard
  • Pick an activation key ranging from F1 to F11
  • Click the 'Create' button on the right side

Scripts Folder

Profiles are stored in the application's 'scripts' folder, in this folder each subfolder represents a profile.
There's also a special script file that gets created upon profile creation called '__events__.py'.
This script file contains functions that get called on different lifecycle events of the profile.
For example 'on_activate' gets called when the given profile gets activated.
These functions can be used to e.g connect to other applications or to setup input blocking, or to do clean up.
