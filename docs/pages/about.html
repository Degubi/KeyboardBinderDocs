Keyboard Binder


About

Keyboard Binder is an application that allows the handling of multiple keyboards separate from each other.
It allows binding scripts to the keyboard's buttons. It also allows blocking keyboard input per keyboard.

Keyboard Binder comes with a bunch of builtin functionality for things like simulating mouse/keyboard interactions or interacting with applications like Premiere Pro or OBS.


Supported Application Integrations

Premiere Pro Twitch
OBS VLC

Requirements

  • Python (at least 3.10)

Installing & Running

  • Unzip KeyboardBinder.zip into a folder
  • Start KeyboardBinder.exe
  • Proceed to Application Settings to start learning & using the application
