Keyboard Input Blocking

It can be useful to block the normal keyboard inputs when pressing keys that run scripts.
Keyboard Binder uses selective input blocking, which means you have to manually block keyboard inputs in processes.

The reasoning for this is that some similar programs that use automatic input blocking causes most games to get the user banned because of the nature of keyboard input blocking.
This of course implies that blocking keyboard input in games is a bad idea because of the high chance of getting banned!


Blocking

To start blocking an application's keyboard input use the 'block_process_keyboard_input' function found in the 'Application' module, like so:

Application.block_process_keyboard_input('Adobe Premiere Pro.exe')

This code blocks the keyboard input in Premiere Pro. The keyboard which input gets blocked is the current profile's keyboard.

Note: you have to call this function once per session, so calling it on profile activation is a perfect place.


Unblocking

Unblocking is done automatically when the profile gets disabled, or when Keyboard Binder closes.
