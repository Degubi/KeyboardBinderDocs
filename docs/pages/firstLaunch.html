First Launch

Upon the first launch of the application you'll be asked to create a new profile.
A profile's parameters are:

  • A user defined name
  • A user picked keyboard
  • A user picked key that activates the profile ranging from F1 to F11

This means that there can be multiple profiles created, which means that there can be multiple profiles active at the same time assigned to different keyboards.
