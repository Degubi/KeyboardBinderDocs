Device Setup

To start adding devices used as input devices click on the 'Devices' menu on the left.


Adding Keyboard devices

  • Select 'Add Keyboard Device' on the left
  • Enter the keyboard's name
  • Select a physical keyboard by following the instructions on the screen after clicking on the 'Pick Physical Keyboard' button
  • Click the 'Add' button

Adding Mobile Devices

  • Install the corresponding installer to the device from the 'Mobile' folder
  • Select 'Add Mobile Device' on the left
  • Scan the QR code with the Keyboard Binder mobile application by selecting the 'Scan Code' option from the top menu
  • Click the 'Add' button
