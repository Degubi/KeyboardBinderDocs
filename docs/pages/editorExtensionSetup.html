Editor & Extension Setup

In Keyboard Binder everything is written as Python scripts.
It is recommended to use an editor like Visual Studio Code for a better scripting experience.


Starting Keyboard Binder

Start Keyboard Binder if you haven't already. On the first start it will handle some setup stuff.
After completing the setup it will run in the background. To know if it is running you can check the application's green keyboard tray icon:


Before starting using keyboards it might be useful to run code without all of the keyboard hassle.


Visual Studio Code Support

It is possible to run code in Keyboard Binder from VSCode using an extension:

  • Install the Python extension from the extension store
  • Install the KeyboardBinder-Debugger extension from the KeyboardBinder application folder (In VSCode 'Install from VSIX' and browse for the 'KeyboardBinder-Debugger.vsix' file)
  • Change the language to Python in the bottom right corner
