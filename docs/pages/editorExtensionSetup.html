Editor & Extension Setup

In Keyboard Binder everything is written as Python scripts.
It is recommended to use an editor like Visual Studio Code for a better scripting experience.


Visual Studio Code Support

It is possible to run code in Keyboard Binder without the keyboard hassle from VSCode using an extension:

  • Install the Python extension from the extension store
  • Install the KeyboardBinder-Debugger extension from the KeyboardBinder application folder (In VSCode 'Install from VSIX' and browse for the 'KeyboardBinder-Debugger.vsix' file)

Note: The Keyboard Binder modules have very good support for Python typings.
Because of this it is also recommended to setup type checking inside the editor for convenience & avoiding some common errors.
