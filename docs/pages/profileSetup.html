Profile Setup

After adding some devices the next step is to create profile:


Adding Profiles

  • Select 'Profiles' menu on the left
  • Enter the profile's name
  • Select a device
  • If using a keyboard device:
    • Pick an activation key ranging from F1 to F11
    • The profile can be manually activated by pressing this key on the keyboard
    • When enabling the 'Activate on Startup' option the profile gets auto activated upon application startup
  • If using a mobile device:
    • The profile can be manually activated by scanning the QR code on the profile's page
    • When enabling the 'Activate on Startup' option the QR code gets shown on startup for activation
  • Click the 'Create' button on the right side

Profiles are stored in the application's 'scripts' folder, in this folder each subfolder represents a profile:

  • scripts
    • Profile 1
      • __events__.py
      • A.py
      • C.py
    • Profile 2
      • __events__.py
      • NP_5.py
      • NP_3.py
