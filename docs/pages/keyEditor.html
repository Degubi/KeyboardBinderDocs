Key Editor

After clicking the key the editor switched to key editing mode.
Each key has a set of assigned commands.
For the list of commands see the Commands menu.

Note: You can have multiple key editor tabs open at the same time.
The top bar of the center panel is a list of tabs of the active editors. The first one is always the profile editor.

Command Format

The format is very simple. Each line in the editor should contain 1 correctly formatted command call.
Each line should look like this: [namespace].[function]([parameters])
You can see the list of available namespaces on the left panel. Open down the 'Mouse' namespace and double click on the 'moveAbs' label.
A line like this has appeared in the editor:

Mouse.moveAbs(x = ?, y = ?)

This inserted line contains the following:

  • The namespace is 'Mouse'
  • The function is 'moveAbs'
  • 2 parameters: 'x' and 'y' both with the value '?'

This command is not yet walid, because the 'x' and 'y' parameters are supposed to be integers, not question marks.
Let's change those question marks to integers, like this:

Mouse.moveAbs(x = 50, y = 50)

This line is now a valid command.
To save a key right click on the key's tab and hit 'Close'
